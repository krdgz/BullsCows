# Toros y Vacas – React

Una implementación moderna del juego clásico “Toros y Vacas” construida con React y Dexie.
Además es na PWA (Progressive Web App)

Dos jugadores definen números secretos de cuatro cifras y alternan turnos para adivinar el número del contrario. La aplicación gestiona turnos, resultados por intento (toros y vacas), historial por jugador, modal de victoria y flujo de nuevo juego.

Pueden probarla con este link: https://torosyvacas.netlify.app/
